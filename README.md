# Penicillin Yield Predictor

## Setup
1. Put your 3 downloaded Kaggle files in this same folder:
   - penicillin_ann_model.keras
   - scaler_X.pkl
   - scaler_y.pkl
2. Create a virtual environment and install dependencies:
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
3. Run the app:
   streamlit run app_ann.py
